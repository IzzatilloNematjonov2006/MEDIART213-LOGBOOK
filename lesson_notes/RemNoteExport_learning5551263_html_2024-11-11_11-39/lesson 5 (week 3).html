  • CSS units:
  • Absolute units:
    • px (pixels): A pixel is the most commonly used unit in CSS and is based on the display's pixel grid. It represents a fixed size.
    • cm (centimeters): Represents a length in centimeters.
    • mm (millimeters): Represents a length in millimeters.
    • n (inches): Represents a length in inches.
    • pt (points): Typically used in print design, 1 point is equal to 1/72 of an inch.
    • pc (picas): A pica is another print measurement, where 1 pica equals 12 points.
  • Relative units
    • % (percentage): Defines sizes relative to the size of the parent element.
    • em: Represents the size relative to the font size of the parent element.
    • rem (root em): Similar to em, but relative to the font size of the root (<html>) element rather than the parent.
    • vw (viewport width): Represents a percentage of the viewport's (browser window) width.
    • vh (viewport height): Represents a percentage of the viewport's height.
    • vmin and vmax: Represents a percentage of the smaller (vmin) or larger (vmax) dimension between the viewportâ€™s width and height.
    • ch: Represents the width of the "0" (zero) character in the current font. Useful for defining widths for text-heavy elements like input fields.
      • Example: width: 20ch; (Sets the width to 20 characters.)
    • ex: Represents the x-height of the font, typically the height of lowercase letters like "x". This unit is rarely used.
